Linux常用命令

7月 17, 2017

tar

bz2

  • tar cjf X.bz2 files 压缩
  • tar xjf X.bz2 解压

dd

dd if=./{filename} of=/dev/{设备} bs={bytes a time} status=progress

lsof

# lsof -i :[port] 查看端口占用信息

Ubuntu升级单个的软件

apt-get install --only-upgrade <packagename>

合并多个文件

cat f1 f2 > f
